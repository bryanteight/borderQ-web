# BorderQ.com - Frontend

The modern, high-performance web dashboard for real-time border wait time predictions and AI-driven travel insights.

## ðŸš€ Overview

BorderQ provides a premium, user-friendly interface for monitoring US-Canada border wait times. Built with **Next.js 14+**, it features real-time data fusion, interactive trend analysis, and SEO-optimized historical landing pages.

### Key Features
- **Live Status Dashboard**: Real-time wait times for major crossings (Peace Arch, Pacific Highway, Lynden).
- **AI-Driven Insights**: Smart summaries and travel advice generated by the backend RAG (Retrieval-Augmented Generation) engine.
- **Trend Visualization**: Interactive charts powered by **Recharts** to visualize historical and predicted traffic flow.
- **SEO Optimized**: Dynamic routes (`/stats/[port]/[day]`) with Server-Side Rendering (SSR) and JSON-LD schema for search engine visibility.
- **Premium Aesthetics**: A clean "Smart Travel Companion" design system using Tailwind CSS 4, smooth animations with Framer Motion, and high-quality iconography from Lucide React.
